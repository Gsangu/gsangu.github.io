Laravel 5 常用命令

发表于 2017-08-05 | 分类于 PHP





































































创建命令
创建模型：php artisan make:model Test
创建控制器：php artisan make:controller TestController
创建资源控制器：php artisan make:controller TestController –resource
创建资源控制器附带模型php artisan make:controller TestController –resource –model=Test
创建表单验证类php artisan make:request StoreBlogPost（验证名称）
创建数据库迁移php artisan make:migration create_users_table
路由清理缓存php artisan route:clear
其他
查看所有路由php artisan route:list
路由缓存php artisan route:cache
路由清理缓存php artisan route:clear
使用测试用服务器php artisan serve (–port=端口)
数据填充：faker
进入调试php artisan tinker
数据创建factory(Model::class,10)->make()
数据填充factory(Model::class,10)->create()
封装常用——$.extend()方法
使用webpack + vue 简单构建前端项目
